## Overview
Fitness Buddy is a student Android application built using *Kotlin*.  
The app helps users manage basic fitness-related activities in a simple and clean interface.

This project demonstrates my ability to build *real Android applications* with multiple screens, validation, data storage, and UI components.

---

## Features
- Login & Signup with input validation
- Home dashboard screen
- BMI Calculator
- Meal Plan list using RecyclerView
- Workout Timer / Countdown
- SharedPreferences for saving user data
- Custom dialogs and pop-ups
- Clean, user-friendly UI

---

## Technologies Used
- Kotlin
- Android Studio
- XML layouts
- RecyclerView
- SharedPreferences
